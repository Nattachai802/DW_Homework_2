⚙️ การติดตั้ง Apache Superset บน WSL

คู่มือนี้จะแนะนำขั้นตอนการติดตั้ง Apache Superset ซึ่งเป็นเครื่องมือ Business Intelligence (BI) แบบ Open-source สำหรับการสร้าง Dashboard และแสดงผลข้อมูล โดยจะติดตั้งบน WSL (Ubuntu)

ขั้นตอนที่ 1: เตรียมความพร้อมของระบบ

ติดตั้งแพ็กเกจที่จำเป็นสำหรับ Python และการคอมไพล์ต่างๆ รวมถึงอัปเดตระบบให้เป็นปัจจุบัน

sudo apt update && sudo apt install build-essential libssl-dev libffi-dev python3-dev python3-pip python3-venv -y

จากนั้น อัปเกรด pip, setuptools, และ wheel ให้เป็นเวอร์ชันล่าสุด

pip install --upgrade pip setuptools wheel
pip install marshmallow==3.20.1

ขั้นตอนที่ 2: สร้าง Virtual Environment (แนะนำ)

การสร้างสภาพแวดล้อมเสมือน (Virtual Environment) จะช่วยแยกโปรเจกต์และ dependencies ต่างๆ ออกจากกัน ทำให้ง่ายต่อการจัดการ

# สร้างโฟลเดอร์ venv
python3 -m venv venv

# เปิดใช้งาน virtual environment
source venv/bin/activate
หมายเหตุ: หากเปิดใช้งาน venv สำเร็จ จะมีคำว่า (venv) แสดงอยู่หน้า command prompt

ขั้นตอนที่ 3: ติดตั้งและตั้งค่า Superset

ทำการติดตั้ง Apache Superset ด้วยคำสั่ง pip

pip install apache-superset

จากนั้น สร้าง SECRET_KEY ที่จำเป็นสำหรับความปลอดภัย โดยสร้างไฟล์ superset_config.py

nano ~/superset_config.py

เพิ่มโค้ดด้านล่างนี้ลงในไฟล์ แล้วกด Ctrl+X > Y > Enter เพื่อบันทึก (สามารถเปลี่ยน 'YourSecretKey' เป็นข้อความลับของคุณเองได้)

import os
SECRET_KEY = os.getenv('SUPERSET_SECRET_KEY', 'YourSecretKey')

ตั้งค่า Environment Variable เพื่อให้ Superset รู้จักไฟล์ config นี้ และทำการโหลดซ้ำ

echo "export SUPERSET_CONFIG_PATH=~/superset_config.py" >> ~/.bashrc
source ~/.bashrc

ขั้นตอนที่ 4: เตรียมฐานข้อมูลและสร้างผู้ดูแลระบบ

คำสั่งนี้จะทำการสร้างและอัปเกรดฐานข้อมูลภายในของ Superset (ใช้ SQLite เป็นค่าเริ่มต้น)

superset db upgrade

สร้างบัญชีผู้ดูแลระบบ (Admin) โดยทำตามขั้นตอนที่ปรากฏบนหน้าจอ

superset fab create-admin

ทำการโหลดข้อมูลตัวอย่างและสร้าง Role พื้นฐาน

superset init

ขั้นตอนที่ 5: เปิดใช้งาน Superset

รัน Web Server ของ Superset โดยใช้พอร์ต 8088 และอนุญาตให้เข้าถึงจากภายนอก WSL ได้

superset run -h 0.0.0.0 -p 8088 --with-threads --reload --debugger

เมื่อเซิร์ฟเวอร์ทำงานแล้ว ให้เปิดเว็บเบราว์เซอร์บน Windows แล้วเข้าไปที่ URL: http://localhost:8088 จากนั้นล็อกอินด้วยบัญชี Admin ที่คุณสร้างไว้ในขั้นตอนก่อนหน้า
